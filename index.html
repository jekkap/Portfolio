Defuse The Bomb (WIP)

A Whack-A-Mole styled game that replaces moles with bombs. When the start button is pressed the game with start with a bomb being fused. Clicking a bomb will defuse it, igniting a random bomb elsewhere. The goal of the game is to get the highest score.
